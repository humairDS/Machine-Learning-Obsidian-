Bias helps shift the output and gives the model flexibility.

Without bias, the model is forced to pass through the origin (0,0).

**Formula**
```
z=∑(xi​wi​)+b
```

### Example

Weighted Sum:
```
weighted Sum = 3.4
```

Bias 
```
bias = 0.6
```

Then:
```
z = 3.4 + 0.6
z = 4.0
```

Bias acts like an intercept in Linear Regression.

### Analogy

Think of:

```
Output = Salary
```

Even with zero experience, a person may still get a base salary.

That base amount behaves like a bias.
