## What is Linear Algebra?

Linear Algebra is the math of:

- [[Vectors]] → data
- [[Matrices]] → tables of numbers
- [[Transformations]] → changing data
- [[Spaces & dimensions]] → features in ML

It is the backbone of:

- Machine Learning
- Deep Learning
- Computer Vision
- NLP
- Recommendation Systems
