- [[Agglomerative (bottom → top)]]
- [[Divisive (top → bottom)]]

### Example Algorithms

- Hierarchical Clustering
- Agglomerative Clustering

### Best For

- Small to medium datasets
- Understanding relationships between clusters

### Output

Produces a dendrogram (tree structure).

# Important Concept: Linkage

Linkage decides how distance between clusters is measured.

## Types of Linkage

## 1. Single Linkage

Uses minimum distance.
```
Closest Points Between Clusters
```

## 2. Complete Linkage

uses maximum distance
```
Farthest points between clusters
```


## 3. Average Linkage

uses average distance

## 4. Ward Linkage

most popular 
Minimizes variance inside clusters
