
Weights determine the importance of each input.
### Example
Predicting house price:

|Feature|Weight|
|---|---|
|Size|0.8|
|Bedrooms|0.4|
|Age|-0.2|

Here:

- Size has the strongest impact.
- Age negatively affects price.

### Mathematical Example

Inputs:

```
x1 = 2
x2 = 3
```

Weights:

```
w1 = 0.5
w2 = 0.8
```

Calculation:

```
(2×0.5)+(3×0.8) 
=1+2.4
=3.4
```
The weighted sum is **3.4**
